### Requirements to run
1. sqlite Database. Download [here](https://sqlite.org/download.html)

### Steps to run
1. go get github.com/mrfojo/go-forex/src/cmd/
2. cd $GOPATH/src/github.com/mrfojo/go-forex/src/cmd/
3. go run main.go

### Testing
1. cd src/gotest/
2. go test
